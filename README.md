# 🏥 Hospital Patient Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing hospital patient data to generate meaningful business insights using **Python, Excel, and Power BI**. The workflow involved data cleaning, preprocessing, exploratory analysis, and building an interactive dashboard to help hospital management monitor key performance indicators.

---

## 🎯 Project Objectives

- Clean and preprocess raw hospital patient data.
- Analyze department-wise revenue and patient distribution.
- Evaluate doctor performance.
- Monitor patient feedback across departments.
- Identify top revenue-generating cities.
- Analyze monthly admission trends.
- Build an interactive Power BI dashboard for business decision-making.

## 📂 Project Workflow

### 1. Data Cleaning (Python)

Performed the following preprocessing steps using Pandas:

- Imported dataset
- Removed duplicate records
- Handled missing values
- Corrected inconsistent values
- Standardized text columns
- Converted date columns into DateTime format
- Created Length of Stay feature

---

### 2. Data Validation & Cleaning (Excel)

Further validated the dataset using Excel by:

- Removing duplicates
- Checking missing values
- Correcting spelling mistakes
- Standardizing categorical values
- Verifying date formats
- Ensuring data consistency

---

### 3. Data Transformation (Power Query)

Performed additional transformations in Power BI:

- Changed data types
- Removed blank rows
- Trimmed extra spaces
- Cleaned text values
- Replaced inconsistent values
- Created calculated columns
- Loaded cleaned data into the data model
